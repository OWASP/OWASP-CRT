# OWASP-CRT Database Branch

This branch is automatically managed by GitHub Actions and serves as the storage database for the **OWASP Community Recognition Tool**.

It contains dynamically generated JSON files for:
- Issued contributor certificates (`/certs`)
- Rate limiting and cooldown attempts (`/attempts`)

**Important Notice:** 
Please **do not** manually edit files or open Pull Requests against this branch. All data is handled programmatically by our IssueOps workflow. Any manual changes will be overwritten or rejected.

Looking for the source code or want to contribute? Please visit the [main branch](../../tree/main).
